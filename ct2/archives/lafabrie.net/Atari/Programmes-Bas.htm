Dans cette rubrique, vous trouverez tous les programmes que j'ai écrit pour les ordinateurs Atari.
 
 

J'ai débuté la programmation en C comme autodidacte voici cinq ans. J'utilise PURE C pour la programmation en C et INTERFACE 2.33 pour la création des interfaces au format RSC. Le fait de ne pas incorporer les fichiers RSC dans le code de mes programmes permet de les traduire facilement dans différentes langues. Les bonnes volontés sont les bienvenues pour les traductions.
