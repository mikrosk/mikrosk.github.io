Installation d'Icon Extract
 
Elle est automatisée. Elle se déroule en six ou sept phases (si vous effectuez une installation sur une ancienne version) qui vous permettrons de choisir les modules que vous désirez installer ainsi que la partition du disque dur de destination.

Version du module d'installation : 1.2 du 20 septembre 2004

  
 
Phase n°1 :
Début de l'installation et explications sur le processus qui va suivre.


Phase n°2 :
Présentation de la licence du logiciel. Le bouton OUI restera grisé tant que vous n'en n'aurez pas lu l'intégralité

.
  
   
Phase n°3 :
Choix du répertoire d'installation et des modules additionnels. Le bouton SUIVANT restera grisé tant que le répertoire choisi ne disposera pas d'un espace libre nécessaire.

  
Phase n°3 bis (effacement):
Effacement du répertoire d'une ancienne version et récupération des informations contenues dans votre clef d'enregistrement.
Elle est réutilisée pour sérialiser la nouvelle version du programme. Dans le cas contraire installation de la version de démonstration.
  
   
Phase n°4 :
Insertion de votre clef d'enregistrement pourt sérialiser le programme. Si vous êtes passé par la Phase 3b, votre licence s'affichera automatiquement. Dans le cas contraire installation de la version de démonstration.
  
Phase n°5 :
Installation du programme et des modules choisis dans le répertoire de destination.
  
       
 
Phase n°6 :
Fin de l'Installation. Vous pouvez choisir de consulter le fichier des nouveautés et lancer le programme automatiquement après avoir quitter l'installateur.
