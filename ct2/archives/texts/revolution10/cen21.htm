Speicher

Sie können in ihrem Falcon ihre vorhandene Speicherkarte stecken lassen. Sie sollten aber ein Auge auf dessen Geschwindigkeit haben. Chips mit einer Zugriffszeit von 60ns werden hier Pflicht. Gleiches trifft auf EDO-Simm zu, das sie auf die CENTurbo 2 Karte stecken, sofern sie das nicht gleich mitgekauft haben.
Beachten Sie hier bitte dringend eine flache Bauform, ansonsten wird die Falcon Tastatur auf das EDO-Modul drücken!

Vergessen Sie Speichererweiterungen wie "FalconWings" und ähnliches! CENTEK benutzt den vorhandenen Speicher als "Chip-RAM", Speicher der auf dem Beschleuniger als EDO bereitgestellt wird fällt ohne Abstriche komplett in den "FAST-RAM" Bereich und kann so entsprechend genutzt werden. Beachten Sie, daß bei zu langsamen "Chip-RAM" Betriebsstörungen auftreten können. Beachten Sie bitte dringlichst, daß CENTurbo2 ein wirklich schnelles Fast-RAM (EDO Simm) mit einer Zugriffzeit von 60ns benötigt, um die vollen Geschwindigkeiten entwickeln zu können!
Normale PS2-Module können nicht verwendet werden. Im Turbomodus kann man beim Booten beide Speicherbereiche anzeigen lassen.
