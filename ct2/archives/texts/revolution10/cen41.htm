Sie können die Bildschirmauflösungen jederzeit an der gewohnten Stelle im Desktop wechseln. Anstelle der bekannten Dialogbox von Atari, erscheint einfach der CENTSCREEN-Dialog. Von diesem können sie dann in jede gewünschte Auflösung und Farbtiefe umschalten.

SETUP

Enthält die französische Beschreibung des Menüs, das sie zu Gesicht bekommen, wenn Sie ihren Falcon im Turbomodus gebootet haben. Neben dem Text finden Sie die einzelnen Menüs als .GIF Bild.
