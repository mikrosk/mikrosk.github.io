Die Kompatibilität:

Es existiert unter www.centek.fr eine laufend aktualisierte Liste von Software, die bereits erfolgreich unter DOLMEN bzw. mit dem Beschleuniger getestet werden konnte. Die Liste ist keinesfalls komplett. Beachten Sie bei der Software die Unterschiede, ob
diese unter TOS, oder nur in bestimmten Modi läuft.
Normalerweise sollte allerdings all das laufen, was sich an die elementaren Regeln der GEM-Programmierung gehalten hat.
Wenn Sie eine Software haben, die von Anfang an als "TT-kompatibel" eingestuft ist, können Sie davon ausgehen das dieses Programm auch unter DOLMEN / TOS 7 laufen wird.
Software die als "Falcon only" deklariert ist, sollte auch nur in einem Falcon-Modus betrieben werden. In erster Line wird es eine Reihe von Audio-Programmen geben, die weder das Fast-RAM noch den Turbo-Modus verkraften; weiterhin sollte aber auch das im normalen Modus laufen.

CENTurboII und dessen Betriebsysteme können kein Wasser teilen und auch keine Wunder aus alter Software erzeugen oder herbei beten. :-)
